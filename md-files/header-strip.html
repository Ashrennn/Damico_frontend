Header Strip Component - Developer Guide

🎨 Design System & Color Scheme

  • Primary Background: #011A3F (Dark Navy Blue)
  • Text & Icons: #D7E2FE (Light Blue-Gray)
  • Glass Morphism: rgba(255, 255, 255, 0.1) with backdrop-filter: blur(15px)

🏗️ Layout Structure

  • Fixed Header: position: fixed, top: 0, height: 60px
  • Five Equal Sections: flex: 1 for equal distribution
  • Section A Sub-division: Three equal sub-containers for social icons
  • Debug Borders: Colored borders for development visibility

🎯 Interactive Elements

Neomorphism Style (All Buttons)

  • Background: linear-gradient(145deg, #011A3F, #001a2f)
  • Shadows: box-shadow: 3px 3px 6px #000000, -3px -3px 6px #4a5568
  • Active State: inset shadows with scale(0.95) transform
  • Border Radius: 25px for buttons, 50px for dropdown items

Social Icons (Section A)

  • Style: Circular neomorphism with lighter background
  • Background: linear-gradient(145deg, #D7E2FE, #c5d1f0)
  • Text Color: #011A3F (darker color for contrast)
  • Icons: 🔗 (LinkedIn), 📺 (YouTube), 📝 (Blogger)

Menu Buttons (Sections B, C, D)

  • Labels: "Knowledge Center", "International", "News & Media"
  • Style: Elongated neomorphism buttons
  • Text Color: #D7E2FE (lighter color)

Language Button (Section E)

  • Text: "داميكو" (Arabic for Damico)
  • Style: Elongated neomorphism with dropdown

📋 Dropdown System

Glass Morphism Effect

  • Background: rgba(255, 255, 255, 0.1)
  • Blur: backdrop-filter: blur(15px)
  • Border: 1px solid rgba(255, 255, 255, 0.2)
  • Border Radius: 20px for container, 50px for items

Dropdown Items

  • Layout: Flexbox with space-between for text and icons
  • Text Color: #011A3F (darker color)
  • Hover Effects: Background color change + text/icon movement
  • Animation: translateX(5px) scale(1.05) for text, translateX(-5px) scale(1.05) for icons

Separators

  • Style: Gradient separators with darker color
  • Animation: Expand from center on hover, opacity transition
  • Color: rgba(1, 26, 63, 0.3-0.9) gradient

🔧 JavaScript Functionality

  • Toggle System: toggleDropdown(dropdownId) function
  • Click Outside: Closes all dropdowns when clicking outside
  • Button Detection: Handles both .language-btn and .menu-btn elements

📱 Responsive Considerations

  • Flexbox Layout: Equal section distribution
  • Fixed Positioning: Header stays at top during scroll
  • Z-index Management: Proper layering for dropdowns

🎭 Animation Details

  • Transition Timing: 0.3s ease for smooth animations
  • Hover Effects: Click-only active states (no hover shadows)
  • Text Movement: Only text moves, background stays static
  • Separator Animation: Center-to-edges expansion on hover

🔍 Debug Features

  • Section Borders: Red, Green, Blue, Yellow, Magenta for easy identification
  • Sub-container Borders: Light red for Section A subdivisions
  • Header Border: Red border around entire header strip
